# Delete Webhook
**URL:** `https://api-docs.s18.myenergi.net/operations/deleteWebhook`

**Endpoint:** `DELETE https://api.s18.myenergi.net/webhooks/{webhookId}`

This endpoint allows you to delete a webhook subscription.

> [!CAUTION]
> **This operation is irreversible.**
> Deleting a global webhook will also delete all device webhook subscriptions associated with it.
