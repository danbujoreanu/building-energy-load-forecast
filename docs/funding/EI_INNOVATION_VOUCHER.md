**Scheme:** Innovation Vouchers
**Funder:** Enterprise Ireland (EI)
**Sparc Energy status:** Pre-incorporated (CRO name check done) — **must incorporate before applying**

## Scheme Overview

| Parameter | Value |
|-----------|-------|
| Standard voucher value | €10,000 |
| Co-funded voucher value | €20,000 (EI €10K + company €10K) |
| Call frequency | Open year-round (rolling applications) |
| Maximum per company | 4 vouchers lifetime |
| Eligible spend | R&D with an approved Knowledge Provider (college, IoT, academic institution) |
| Company requirement | Limited company, registered in Ireland, fewer than 50 employees |
| Activity | Must demonstrate new learning / R&D activity — not consultancy |

## Framing for Sparc Energy

**DO NOT frame as:** "completing the MSc thesis" or "deploying existing LightGBM model."

**Frame as:** One of:
1. **Commercial adaptation of academic research** — "Adapting H+24 day-ahead load forecasting for commercial building portfolios in the Irish market, including SEM/SEMO integration, Irish grid constraints (DS3), and multi-building generalisation."
2. **IoT sensor integration for demand-response** — "Developing real-time sensor integration (Ecowitt, myenergi) with automated demand-response control for SME and residential deployment."
3. **Explainable AI for energy compliance** — "Building EU AI Act–compliant transparency layer for AI-driven energy recommendations: P10/P90 confidence display, audit logging, human override interface."

All three are defensible as new R&D because the MSc thesis used Norwegian building data; Irish commercial deployment is genuinely new.

## Application Process

1. **Incorporate Sparc Energy Ltd** (CRO — name check done, costs ~€50–100 via CRO online or €300 via solicitor)
2. **Identify Knowledge Provider** — get provisional agreement from KP before applying
3. **Submit application** at [enterprise-ireland.com/innovation-vouchers](https://www.enterprise-ireland.com/en/innovation/companies/innovation-voucher/) — rolling, no deadline
4. **Receive approval** (typically 4–6 weeks)
5. **Commence R&D** — work must be new, documented, R&D in nature
6. **Submit completion report** — EI audits spend; keep all receipts and meeting notes
