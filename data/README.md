# Data

This directory contains raw building energy data for two Norwegian cities.

## 📖 Detailed Datasets Description
For comprehensive, scientific-level documentation of both datasets, the raw source files are located in `data/Detailed Datasets description`:
- **Drammen:** `Drammen...Scientific_Data.pdf`
- **Oslo:** `Oslo_README.txt`

### 🔑 Unified Code Dictionary (Drammen & Oslo)
Both datasets follow the same metadata structure and column naming convention established by the COFACTOR/SINTEF projects:

**Metadata Headers:**
- `Building_category`: `Sch` (School)
- `Energy_label`: `A-G` (Efficiency rating)
- `Central_heating_system`: `0` (No), `1` (Yes), `2` (Unknown)
- `_heat_source` types: `EB` (Electric boiler), `DH` (District heating), `ASHP` / `GSHP` (Air/Ground source heat pump)

**Time-Series Columns:**
- `Electricity_Imported_Total_kWh` or `ElImp`: Target Variable (Imported grid electricity)
- `Tout`: Outdoor temperature (°C)
- `SolGlob`: Global Solar Horizontal Radiation (W/m2)
- `WindSpd`: Wind speed (m/s)

## Drammen (included)

45 public school and kindergarten buildings from the COFACTOR project.

| Property | Value |
|----------|-------|
| Buildings | 45 (schools & kindergartens) |
| Resolution | Hourly |
| Period | ~2018–2024 |
| Format | `.txt` (semicolon-delimited, metadata header + time-series) |
| Size | ~98 MB total |

**Files are included in this repository** in `data/raw/drammen/`.

**Citation:**
> Cofactor Project, Norway. Building energy data from 45 Drammen public buildings.

## Oslo (not included — download required)

48 public school buildings from SINTEF (Oslobygg KF).

| Property | Value |
|----------|-------|
| Buildings | 48 schools |
| Resolution | Hourly |
| Period | 1–11 years per building |
| Format | `.csv` (same metadata header convention) |
| DOI | [10.60609/2hvr-wc82](https://data.sintef.no/product/dp-679b0640-834e-46bd-bc8f-8484ca79b414) |

**To download:**

```bash
# Download instructions
python scripts/download_data.py --dataset oslo

# Or manually:
# 1. Visit https://data.sintef.no/product/dp-679b0640-834e-46bd-bc8f-8484ca79b414
# 2. Download all 48 CSV files
# 3. Place them in data/raw/oslo/
```

**Citation:**
> Synne Krekling Lien et al. (2025). Hourly Sub-Metered Energy Use Data from 48 Public
> School Buildings in Oslo, Norway. *Data in Brief*.
> License: CC BY 4.0

## Generated files (not in git)

The following files are **generated by the pipeline** and excluded from version control
(they can be large, 400 MB+):

```
data/processed/
    metadata.parquet          ← Building characteristics
    timeseries.parquet        ← Merged hourly time-series
    model_ready.parquet       ← Cleaned + calendar features
    splits/
        X_train_fs.parquet    ← Feature-selected training features
        X_val_fs.parquet
        X_test_fs.parquet
        y_train.parquet       ← Target variable splits
        y_val.parquet
        y_test.parquet
        scaler.pkl            ← Fitted StandardScaler
```

Regenerate by running:
```bash
python scripts/run_pipeline.py --city drammen --stages eda features
```
